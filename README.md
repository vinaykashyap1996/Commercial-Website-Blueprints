## What Is This?

A collection of **complete, production-ready tutorial workflows** that teach you how to build commercial-grade websites and apps using AI tools like Claude, ChatGPT, Cursor, and v0.

Each blueprint includes:

- **AI Prompts** — The exact prompt chains used to generate each part of the project
- **Starter Code** — Clean, commented skeleton code you build on top of
- **Step-by-Step Tutorial** — Detailed walkthrough from empty folder to finished product
- **Deployment Guide** — Go live on Vercel, Netlify, or your own server
- **Commercial Value Estimate** — What clients and agencies typically pay for this type of work

> **This is not a list of links.** Every blueprint is a self-contained, end-to-end tutorial you can follow from start to finish.

## Quick Start

1. Browse the [Blueprint Categories](#blueprints) below
2. Pick a project that matches your skill level
3. Open the blueprint's `BLUEPRINT.md` for the full tutorial
4. Use the AI prompts in the `prompts/` folder with your preferred AI tool
5. Deploy using the included deployment guide
6. **Add it to your portfolio or sell it to a client**

## How Each Blueprint Works

Every blueprint follows the same structure so you always know what to expect:

```
blueprint-name/
├── README.md          # Overview, preview, tech stack, live demo link
├── BLUEPRINT.md       # The full step-by-step build tutorial
├── prompts/           # Numbered AI prompt files
│   ├── 01-setup.md
│   ├── 02-core.md
│   ├── 03-polish.md
│   └── 04-deploy.md
├── starter-code/      # Minimal skeleton files to begin with
└── screenshots/       # Visual references for each step
```

1. **Read the `README.md`** — understand what you're building and the tech involved
2. **Follow `BLUEPRINT.md`** — step-by-step guide walks you through the entire build
3. **Use the prompts** — paste each numbered prompt into your AI tool (Claude, ChatGPT, Cursor)
4. **Check your work** — compare against the screenshots after each step
5. **Deploy** — follow the deployment section to go live
